# 📖 O Grimório do Deserto: A Queda do Quinto Trevo — Wiki

Uma documentação interativa e imersiva (Wiki) construída para o cenário de RPG de mesa **"O Grimório do Deserto"**. Desenvolvido em formato *Single Page Application (SPA)*, este compêndio documenta a lore, personagens, regras de sistema e possíveis finais da campanha, tudo envolto em uma estética retrô e sombria.

## ✨ Características do Projeto

* 🌌 **Estética Dark Pixel Art**: Interface construída com paletas contrastantes (Obsidiana/Carmesim para o Vácuo, e Dourado/Branco para o Reino).
* 🚀 **Single Page Application (SPA)**: Navegação instantânea e fluida entre os capítulos da lore sem recarregamento da página.
* 🎬 **Animações CSS Imersivas**: Efeito CRT (*scanlines*), grimório flutuante com pulsação de energia, efeitos de "glitch" e barras de status dinâmicas.
* 📱 **Totalmente Responsivo**: O layout adapta-se perfeitamente de monitores ultrawide a telas de smartphones, com um menu lateral otimizado (`hamburger menu`) para mobile.
* 🖋️ **Tipografia Retrô**: Uso de fontes temáticas do Google Fonts (`Press Start 2P`, `VT323` e `Cinzel Decorative`).

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído puramente com tecnologias front-end nativas, visando leveza e facilidade de manutenção:

* **HTML5**: Estrutura semântica e acessível.
* **CSS3**: Variáveis globais (`:root`), Grid/Flexbox, `keyframes` e media queries para responsividade.
* **JavaScript (Vanilla)**: Lógica limpa para manipulação do DOM na troca de abas e abertura de menus no formato SPA. Nenhuma biblioteca externa ou framework foi utilizado.

## 📚 Estrutura do Compêndio

A Wiki está dividida em 6 capítulos principais que narram a progressão do universo e do sistema:

1. **O Prólogo:** A introdução ao Reino de Aethelgard e o conflito inicial entre os irmãos Dante e Elias.
2. **Personagens:** *Cards* detalhados com estatísticas interativas dos protagonistas (O Vazio vs. A Lâmina Dourada).
3. **O Grimório & A 5ª Folha:** A história do Dia da Ascensão e o pacto sinistro com Astaroth.
4. **O Mundo:** O mapeamento político e cosmológico (A Facção do Vácuo, O Pilar do Mundo, etc).
5. **Mecânicas de Sistema:** Regras cruciais para o Mestre de Jogo (Pontos de Inércia e Absorção de Magia).
6. **Finais da Campanha:** Condições narrativas baseadas nas escolhas dos jogadores (Final de Vilão vs. Final de Redenção).
